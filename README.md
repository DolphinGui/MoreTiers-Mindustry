# MoreTiers
Exactly as it sounds like, more tiers for various production blocks. Also more interactions with liquids.

## Importing

Simply download this repository as a zip, then import it through the `Mods` dialog in Mindustry. Or, unzip this repo inside Mindustry's `mods/` folder.

## Contributing

I won't regularly check on pullrequests, so don't expect miracles. I might grant a push request if it's a bug-fix/or makes the mod more interesting.
